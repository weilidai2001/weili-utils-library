Function mergeTwoListsAsUnique

  • mergeTwoListsAsUnique<T>(list1, list2, uniqueKey): T[]

  • Combines two arrays of objects into a single array, removing duplicates based on a specified key.

    Type Parameters

    • T

    Parameters

    • list1: T[]

      The first array of objects to merge.

    • list2: T[]

      The second array of objects to merge.

    • uniqueKey: keyof T

      The key to use for identifying unique objects.

    Returns T[]

    A new array containing the merged and uniqued objects from both input lists.

    Example

    const list1 = [
        { key: 'apple', value: '' },
        { key: 'banana', value: '' },
    ];
    const list2 = [
        { key: 'apple', value: 'apple' },
        { key: 'date', value: 'date' },
    ];
    const result = mergeTwoListsAsUnique(list1, list2, 'key');
    // Returns: [
    //     { key: 'apple', value: 'apple' },
    //     { key: 'banana', value: '' },
    //     { key: 'date', value: 'date' }
    // ]
