Function partition

  • partition<T>(array, predicate): [T[], T[]]

  • Splits an array into two groups based on a predicate function. The first group contains all items that pass the predicate (return true), and the second group contains all items that fail (return false).

    Type Parameters

    • T

    Parameters

    • array: T[]

      The array to partition.

    • predicate: ((value) => boolean)

      The function used to test each element.

        • (value): boolean

        • Parameters

          • value: T

          Returns boolean

    Returns [T[], T[]]

    A tuple where the first array contains items that passed and the second contains items that failed.

    Example

    const numbers = [1, 2, 3, 4, 5];

    const [evens, odds] = partition(numbers, n => n % 2 === 0);
    console.log(evens); // Output: [2, 4]
    console.log(odds);  // Output: [1, 3, 5]

    Description

    The function iterates through the input array once, testing each item against the predicate, and returns two separate arrays without modifying the original array.
